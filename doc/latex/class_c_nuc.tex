\doxysubsection{Detailed Description}
An AZURE compound nucleus. 

The compound nucleus is the fundamental concept of R-\/\+Matrix theory. As such, the \doxylink{class_c_nuc}{CNuc} object in AZURE is the top level container object for all structure and reaction objects. Specifically, the \doxylink{class_c_nuc}{CNuc} object is the container object for vectors of \doxylink{class_p_pair}{PPair} and \doxylink{class_j_group}{JGroup} objects, within which all other nuclear data objects are contained.
