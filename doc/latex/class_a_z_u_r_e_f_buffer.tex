\doxysubsection{Detailed Description}
A container class for a pointer to a file buffer. 

The \doxylink{class_a_z_u_r_e_f_buffer}{AZUREFBuffer} class contains a pointer to an acutal file buffer, as well as the entrance and exit pair keys to which the file buffer corresponds. ~\newline
