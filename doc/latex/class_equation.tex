\doxysubsection{Detailed Description}
A class for parsing algebraic expressions. 

The \doxylink{class_equation}{Equation} class is used in AZURE to parametrize stopping cross sections as a function of energy. An arbitrary equation can be given, and parameters specified, which is then parsed and evaluated on the fly whenever stopping cross section is needed.
