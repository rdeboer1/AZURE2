\doxysubsection{Detailed Description}
A temporaray T-\/\+Matrix structure. 

The \doxylink{struct_temp_t_matrix}{Temp\+TMatrix} structure is used to coherently add T-\/matrix elements from pathways with like $ J,l,l\textnormal{\textquotesingle} $ values for the calculation of angle integrated cross section. This is primarly used to facilitate the interference between internal and external pathways.
