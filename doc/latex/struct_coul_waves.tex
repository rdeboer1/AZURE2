The return structure of the \doxylink{class_coul_func}{Coul\+Func} function class. 

The \doxylink{struct_coul_waves}{Coul\+Waves} structure contains both the irregular and regular solutions to the Coulomb equation, as well as their derivatives with respect to $ \rho $.
