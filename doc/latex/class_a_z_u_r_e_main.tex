\doxysubsection{Detailed Description}
The top-\/level AZURE function class. 

The \doxylink{class_a_z_u_r_e_main}{AZUREMain} function class is the top level function class in the AZURE package. It is called directly from them \doxylink{_a_z_u_r_e2_8cpp_a0ddf1224851353fc92bfbff6f499fa97}{main()} using the configuration parameters read from the runtime file as well as from the command shell prompt.
