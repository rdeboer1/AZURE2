\doxysubsection{Detailed Description}
An AZURE Particle Pair. 

In R-\/\+Matrix theory, the configuration space in the external region is decomposed into combinations of particle pairs, traditionally given by the symbol $ \alpha $. In AZURE, these particle pair are represented by a \doxylink{class_p_pair}{PPair} object. \doxylink{class_p_pair}{PPair} objects are containers for vectors of \doxylink{class_decay}{Decay} objects.
