\doxysubsection{Detailed Description}
An AZURE internal reaction pathway. 

An \doxylink{class_m_group}{MGroup} in AZURE represents a given entrance and exit channel through a $ J^\pi $ group. These can be visualized as paths entering one row of the T-\/\+Matrix, and exiting through a column.
