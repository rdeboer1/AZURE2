\doxysubsection{Detailed Description}
A function class to calculate the reaction rate. 

The \doxylink{class_reaction_rate}{Reaction\+Rate} function class is used to calculate the reaction rate based on a set of R-\/\+Matrix parameters over a range of stellar temperatures.
