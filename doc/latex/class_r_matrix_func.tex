\doxysubsection{Detailed Description}
A function class to calculate the T-\/\+Matrix using the R-\/\+Matrix. 

The \doxylink{class_r_matrix_func}{RMatrix\+Func} function class calculates the T-\/\+Matrix for a given energy point using the compound nucleus object. The \doxylink{class_r_matrix_func}{RMatrix\+Func} class is a child class of \doxylink{class_gen_matrix_func}{Gen\+Matrix\+Func}, where the cross section is calculated from the T-\/\+Matrix.
