\doxysubsection{Detailed Description}
An AZURE decay pair. 

In AZURE, a \doxylink{class_decay}{Decay} object represents a decay pair of the compound nucleus. The \doxylink{class_decay}{Decay} object is keyed to a particle pair in the \doxylink{class_p_pair}{PPair} vector, and serves as a container class for the \doxylink{class_k_group}{KGroup} and the \doxylink{class_k_l_group}{KLGroup} vectors and their subsequent reaction pathways. ~\newline
