\doxysubsection{Detailed Description}
A function class to calculate the channel integrals in the denominator of the $ N_f^{1/2} $ term. 

The \doxylink{class_n_f_integral}{NFIntegral} class returns the channel integral given by $ \int_a^\infty \left[ \frac{W_c(kr)}{W_c{ka_c}} \right]^2 $.
