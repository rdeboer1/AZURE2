\doxysubsection{Detailed Description}
An AZURE $ J^\pi $ group. 

In R-\/\+Matrix theory, levels are grouped according to their $ J^\pi $ values. There is one R-\//\+A-\/\+Matrix, and thus one T-\/\+Matrix, for each $ J^\pi $ group. A \doxylink{class_j_group}{JGroup} object holds vectors of \doxylink{class_a_level}{ALevel} and \doxylink{class_a_channel}{AChannel} objects.
