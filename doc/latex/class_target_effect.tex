\doxysubsection{Detailed Description}
An AZURE target effect entry. 

Experimential effects including gaussian beam convolution, target integration, and a combination of the two are grouped under the \doxylink{class_target_effect}{Target\+Effect} class. An object is created corresponding to each corresponding entry in AZURESetup2.
