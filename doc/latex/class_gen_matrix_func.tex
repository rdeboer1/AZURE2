\doxysubsection{Detailed Description}
A generalized function class to calculate cross sections. 

The \doxylink{class_gen_matrix_func}{Gen\+Matrix\+Func} function class is the general form of the function used to calculate cross section from R-\/\+Matrix parameters. It is the parent class of \doxylink{class_a_matrix_func}{AMatrix\+Func} and \doxylink{class_r_matrix_func}{RMatrix\+Func}.
