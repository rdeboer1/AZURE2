\doxysubsection{Detailed Description}
A function class for calculating effective charge without long-\/wavelength approximation. 

The \doxylink{class_effective_charge}{Effective\+Charge} class calculates the effective charge needed for external capture without using the long wavelength approximation. The formalism is based on J.\+L. Friar and S. Fallieros, Phys. Rev. C 29, 1645 (1984).
