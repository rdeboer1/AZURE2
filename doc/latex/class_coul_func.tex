\doxysubsection{Detailed Description}
A function class to calculate Coulomb functions for positive energy channels. 

The \doxylink{class_coul_func}{Coul\+Func} function class calculates the solutions to the Coulomb equation, as well as other useful quantities such as shift functions and their energy derivative and penetrabilities.
