\doxysubsection{Detailed Description}
A function class for negative energy shift functions. 

A shift function for negative energy channels is calculated as $ S=\rho \frac{W_c\textnormal{\textquotesingle}(k\rho)}{W_c(k\rho)} $, where the prime indicates the derivative with respect to $ \rho $. The AZURE function class \doxylink{class_shft_func}{Shft\+Func} uses the GSL package to calculates the numerical derivative.
