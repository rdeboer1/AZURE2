\doxysubsection{Detailed Description}
A container class to hold Minuit parameters in AZURE. 

The \doxylink{class_a_z_u_r_e_params}{AZUREParams} class holds the Minuit parameters determined in the fit. The class also has member functions corresponding to reading and writing of the parameters and their errors.
