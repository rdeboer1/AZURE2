\doxysubsection{Detailed Description}
An AZURE $ s,s\textnormal{\textquotesingle} $ group. 

In R-\/\+Matrix formalism, the equations required to calculate the cross section usually nested inside sums over entrance and exit channel spins. For this reason AZURE groups reaction pathways according to their entrance and exit channel spins. Each \doxylink{class_k_group}{KGroup} object is a container for vectors of \doxylink{class_m_group}{MGroup} and \doxylink{class_e_c_m_group}{ECMGroup} objects. ~\newline
