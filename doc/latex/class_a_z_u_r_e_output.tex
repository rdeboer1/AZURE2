\doxysubsection{Detailed Description}
A class to assist in writing AZURE output files. 

The \doxylink{class_e_data_a3e22709dd5e91d963afe84c815e1d742}{EData\+::\+Write\+Output\+Files} function simply loops over all \doxylink{class_e_segment}{ESegment} and \doxylink{class_e_point}{EPoint} objects when writing the output of an AZURE calculation. To ensure that all output for a given entrance and exit pair combination is written to a single file, the \doxylink{class_a_z_u_r_e_output}{AZUREOutput} class is used. The \doxylink{class_a_z_u_r_e_output}{AZUREOutput} object is a container for a vector of \doxylink{class_a_z_u_r_e_f_buffer}{AZUREFBuffer} objects.

The parenthesis operator is defined so that the instance of \doxylink{class_a_z_u_r_e_output}{AZUREOutput} can be called as a function. ~\newline
 The instance is called using a reference to an entrance and exit pair key combination. The function tests if there is a pointer to an \doxylink{class_a_z_u_r_e_f_buffer}{AZUREFBuffer} object in a vector. If such a pointer exists, the pointer to the actual file buffer contained in the corresponding \doxylink{class_a_z_u_r_e_f_buffer}{AZUREFBuffer} object is returned. Otherwise, a new \doxylink{class_a_z_u_r_e_f_buffer}{AZUREFBuffer} object is created with the entrance and exit key, a pointer to that object is stored in a vector, and the pointer to the actual new file buffer is returned.
