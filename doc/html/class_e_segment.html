Detailed Description

An AZURE data segment.

An AZURE data segment is specified by an entrance and exit particle pair key, as well as a range of energy and angle value and a data file name. The segment also contains flags specifing they type of data point it contains. The ESegment object is the container object for a vector of EData objects.
