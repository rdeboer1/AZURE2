Detailed Description

An AZURE Particle Pair.

In R-Matrix theory, the configuration space in the external region is decomposed into combinations of particle pairs, traditionally given by the symbol . In AZURE, these particle pair are represented by a PPair object. PPair objects are containers for vectors of Decay objects.
